How to Use the CDN

You can include CSS, JavaScript, or any other file type directly into your website by referencing the URLs below.

Examples:

1. CSS File

Add this to your HTML to include the CDN-hosted CSS file:

<link rel="stylesheet" href="https://free-cdn.vercel.app/static/css/styles.css">

2. JavaScript File

Add this to your HTML to include the CDN-hosted JavaScript file:

<script src="https://free-cdn.vercel.app/static/js/script.js"></script>

3. Images or Uploaded Files

Use the generated URL after uploading a file. Example:

<img src="https://free-cdn.vercel.app/uploads/example.png" alt="Example Image">
